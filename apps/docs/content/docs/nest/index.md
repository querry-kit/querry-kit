# Query Kit for NestJS APIs

`@querry-kit/nest` brings fields projection, Prisma-style query services, CASL policies, decorators, pipes, pagination DTOs, and object utilities into one focused package.

## Build a resource API without repetitive glue

- **Resource query facade** prepares fields, relation includes, service calls, DTO mapping, and response projection.
- **Prisma and CASL** keep authorization-aware database queries explicit and optional.
- **Fields projection** validates requested DTO fields and returns the response shape a client asked for.
- **NestJS utilities** provide OpenAPI decorators, pipes, pagination DTOs, and parsing helpers from a single import surface.
