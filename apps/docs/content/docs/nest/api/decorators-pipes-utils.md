# Decorators, Pipes, Utilities

This package includes the small NestJS utilities that were previously split into focused packages. The API docs are now grouped by concern:

- [OpenAPI Decorators](/docs/nest/api/openapi) for `ApiResourceQuery`, `ApiPaginatedResponse`, and `ApiErrorResponses`.
- [Decorators](/docs/nest/api/decorators/) for route parameter and DTO property decorators.
- [Pipes](/docs/nest/api/pipes/) for `QueryTransformPipe` and `EmptyStringToNullPipe`.
- [Object Utilities](/docs/nest/api/object-utils) for object parsing, diffs, Decimal serialization, and validation helpers.

The old combined page is kept as a stable entry point for existing links.
